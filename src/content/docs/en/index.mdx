## Features

<CardGrid stagger>
  <Card title="Live Recording" icon="sourcehut">
    Supports live recording/history/preview/monitoring only. <br />
    Currently supported platforms: Douyin/Bilibili/YY/Bigo
  </Card>
  <Card title="Anchor Data" icon="seti:info">
    Supports obtaining anchor homepage data, automatically refreshed daily,
    displaying statistical charts.
  </Card>
  <Card title="File Management" icon="document">
    Supports file listing/detection/online preview.
  </Card>
  <Card title="Message Notification" icon="comment">
    Supports live start/end notifications. <br />
    Currently supported channels: Email/DingTalk/WeChat Work/Feishu/Gotify.
  </Card>
  <Card title="Cloud Drive Upload" icon="puzzle">
    Supports uploading to cloud drives via Openlist.
  </Card>
</CardGrid>
